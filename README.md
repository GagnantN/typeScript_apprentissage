Configuration d'un nouveau environnement pour TypeScript et Node.js

-- Installer Node.js pour Window --
  Allez sur ce site : https://nodejs.org/fr
  Cliquer sur le bouton vert "Obtenir Node.js"
  Clique sur le bouton vert "Window installer (.msi)
  Cliquer sur l'application et faite "Suivant" à chaque fois

-- Installer TypeScript --
  Dans le terminal utiliser cette commande : npm install -g typescript

-- Vérification --
  node -v    // Pour Node.js
  tsc -v     // Pour TypeScript

-- Lancement d'un fichier ts --
  tcs nomFichier.ts
  node nomFichier.js
